HOW TO USE:

Step 1: include the plugin files


<link href="youtube-gallery/yt-gallery.css" type="text/css" rel="stylesheet" />
<script src="youtube-gallery/yt-gallery.js"></script>


Step 2: initiate the plugin without options


<script>
	$(function() {
		$('#gallery').YouTubeGallery();
	});
</script>

#gallery: videos container


Step 3:The HTML markup


<div id="gallery">
	<div class="yt-gallery" data-video-id="[VIDEO ID 1]"></div>
	<div class="yt-gallery" data-video-id="[VIDEO ID 2]"></div>
	<div class="yt-gallery" data-video-id="[VIDEO ID ...]"></div>
	<div class="yt-gallery" data-video-id="[VIDEO ID N]"></div>
</div>


OPTIONAL PARAMETERS:


cover:			[STRING]  thumbnail image (see extras below)
videoInfo:		[BOOLEAN] display date and video author
openInFancybox: 	[BOOLEAN] display video at new page (Youtube) or play in Fancybox
autoplay:		[BOOLEAN] autoplay video (only if openFancybox is true)
texts: {
	post_date:	[STRING]  default text to date,
	post_author:	[STRING]  default text to author
},
messages: {
	error:		[STRING]  default text to error
}


EXTRAS

HOW YOUTUBE THUMBNAIL WORKS

XML	: https://gdata.youtube.com/feeds/api/videos/[VIDEO ID]?v=2
EMBED	: <iframe width="340" height="191" src="//www.youtube.com/embed/[VIDEO ID]" frameborder="0" allowfullscreen></iframe>
