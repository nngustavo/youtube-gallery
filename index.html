About Youtube Gallery

This jQuery plugin displays your Youtube videos as gallery with (or without) Fancybox integration.

You can download the plugin files or clone the repository, like this:

$ git clone https://moreirapontocom@github.com/moreirapontocom/youtube-gallery.git
